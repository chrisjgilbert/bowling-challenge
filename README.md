Bowling Challenge
=================
## How to use  
`open SpecRunner.html` to view the tests  
```
game = new Game();   
game.roll(4);   
game.roll(6);   
game.roll(10);   
game.roll('X')   
```
fill in an 'X' after your strike!   
```
game.roll(4);
game.roll(5);
```
until you have played 10 frames, then   
```
game.score();   
```
to return your score!
